# 🏥 Hospital Management System (Node.js + MySQL + Postman)

## 📌 Project Overview

The Hospital Management System is a Node.js and MySQL-based REST API project that manages hospital information such as doctors, patients, pharmacy, and staff. The project uses Express.js for the backend and MySQL as the database. APIs are tested using Postman.

---

## 🚀 Features

- Doctor Management
- Patient Management
- Pharmacy Management
- Staff Management
- MySQL Database Integration
- REST API using Express.js
- Postman API Testing

---

## 🛠 Technologies Used

- Node.js
- Express.js
- MySQL
- mysql2
- Postman
- JavaScript

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/karthikrchintamani-maker/HOSPITAL_MYSQL_POSTMAN.git
```

Go to project folder

```bash
cd HOSPITAL_MYSQL_POSTMAN
```

Install dependencies

```bash
npm install
```

---

## ▶ Run the Project

```bash
node appoloERP.js
```

Server runs on

```
http://localhost:8000
```
